# 🌪️ APP Epicentro Digital

Alicativo desenvolvido como parte da **Global Solution do 1º semestre de 2025** para a disciplina de **Kotlin**, com o prof. **Ewerton**. O tema do projeto é **Eventos Extremos**, e criamos um app que funciona como um **radar digital** para registrar ocorrências como enchentes, secas, incêndios, entre outros.

O app foi baseado no projeto apresentado em aula (Lista de Compras) e expandido com novas funcionalidades.


O usuário pode cadastrar:
- 📍 Nome do local (cidade ou bairro)
- ⚠️ Tipo do evento
- 🔥 Grau de impacto (leve, moderado, severo)
- 📅 Data do evento
- 👥 Número de pessoas afetadas

## 🛠️ Tecnologias e Componentes Utilizados

- **Kotlin** como linguagem principal
- **Android Studio** como IDE
- **RecyclerView** para exibir dinamicamente a lista de eventos
- **ViewModel + LiveData** para manter os dados atualizados e reativos
- **ComponentActivity** e **AppCompatActivity**
- **Intents** para navegação entre telas (usamos aqui para ir para activity Identificacao Alunos)
- **Validações de entrada** com `Toast`
- **Layouts otimizados com ScrollView, LinearLayout e estilo moderno**

- ## 🧪 Como Testar

1. Clone este repositório
2. Abra no **Android Studio**
3. Conecte um dispositivo Android ou use o emulador
4. Clique em ▶️ *Run*

> O app é compatível com Android 8 (API 26) ou superior.
